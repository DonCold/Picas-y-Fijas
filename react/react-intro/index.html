¿Qué es React?

Es una librería open source de JavaScript para desarrollar interfaces de usuario.

Fue lanzada en el año 2013 y desarrollada por Facebook, quienes también la mantienen actualmente junto a una comunidad de desarrolladores independientes y compañías.

Componentes de React

Los componentes permiten separar la interfaz de usuario en piezas independientes, reutilizables y pensar en cada pieza de forma aislada.

Son como las funciones de JavaScript. Aceptan entradas arbitrarias (llamadas “props”) y retornan elementos de React que describen lo que debe aparecer en la pantalla.

Rendimiento y Flexibilidad

Trabajar con cualquier otro Stack que necesitemos no es un problema.
Al ser una librería y estar enfocada en una sola cosa, es fácil de usar y muy rápido.

Incorpora un Virtual DOM para la optimización del DOM.

React Native

También podemos llevar todo el Poder de React a las aplicaciones móviles.
Teniendo las capacidades de la Plataforma Nativa.

Se utiliza principalmente para desarrollar aplicaciones para Android e iOS

React 360

Basado en Three.js, React 360 permite crear aplicaciones 360 con React.

Aprovechando las tecnologías web y el ecosistema React existente, React 360 tiene como objetivo simplificar la construcción de experiencias 360 multiplataforma.
